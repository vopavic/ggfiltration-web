# GG Filtration Web Redesign

Moderní B2B webová prezentace pro G&G filtration, postavená na Next.js a Tailwind CSS.

## Technologie

- **Next.js 15+** (App Router)
- **Tailwind CSS v4**
- **Lucide React** (Ikony)
- **Framer Motion** (Animace)

## Spuštění projektu

1. Instalace závislostí:

```bash
npm install
```

2. Spuštění vývojového serveru:

```bash
npm run dev
```

Web poběží na `http://localhost:3000`.

## Struktura

- `src/app`: Hlavní stránky (Home, Produkty, Kontakt, [slug])
- `src/components`: UI komponenty a layout (Header, Footer)
- `src/lib`: Konstanty a pomocné funkce
- `public/images`: Loga a obrázky
